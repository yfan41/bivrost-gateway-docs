This chapter has two parts, a glossary and a command format reference:

- [6.1. Glossary](/en/reference/glossary/): definitions and calculation methods for concepts such as Machine OEE Data, Group OEE Data and Group Part Count.
- [6.2. Command Format](/en/reference/command-format/): the format of commands such as PLC Data Task, Precondition, Task Interval, Post-processing ($DA$ / $POST$ / $MOCK$), External PLC Data Task, External Machines, Protected API and Authorized APIs.
