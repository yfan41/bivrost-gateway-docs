---
title: "三、基本参数"
---

| 项目 | 参数 |
| --- | --- |
| 输入电压 | DC 12V（自带电源适配器） |
| 工作温度 | -20℃~+60℃，表面空气流动 |
| 工作湿度 | 0%~95% 非凝结状态 |
| 安全认证 | CE，CCC，FCC Class A，RoHS |
| 产品尺寸 | 144\*126.5\*42mm |
| 整机重量 | 0.8kg |
| 固定方式 | 支持桌面式，导轨式，VESA 壁挂式等固定方式 |
| LAN1 | 默认 IP：192.168.100.1，子网掩码：255.255.0.0 |
| LAN2 | IP 地址自动获取，子网掩码自动获取 |
| WIFI | 默认开启，详见 [5.7.2. 无线网配置](/usage/network/#wireless) |
| 通电自动开机 | 默认开启 |
| 开机自动运行采集 | 默认开启 |
| 网关登录地址 | `http://iotgw.local` 或网关 IP 地址 |
| 登录用户名密码 | 默认均为 admin |
